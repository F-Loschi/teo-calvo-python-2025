## 📌 Como utilizar
1. Clone o repositório:
 ```bash
   git clone https://github.com/F-Loschi/teo-calvo-python-2025.git
```

2. Entre na pasta do projeto:
```bash
cd nome-do-repo
```

3. Execute os scripts Python:
```bash
python nome_do_arquivo.py
```
